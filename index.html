Flobot Overview

What?

Flobot is a low-code development framework for Node.js.

Why?

Flobot has been developed to offer an alternative for organizations (especially non-profits and Government departments) who need modern and continually-evolving back-office software - but can do without all the complexity, vendor lock-in and expense that traditionally comes with it.

By placing an emphasis on openness, collaboration and self-sufficiency, Flobot offers a service that has been specifically engineered to meet the challenges facing today’s Public Sector.

Built with at West Midlands Fire Service

Installation

Once you’ve installed Node.js (at least v6.9 LTS), then Flobot can be downloaded via npm:

npm install flobot --save

Features

  • Encapsulate business functions in low-code “Blueprints” - great for interoperability and developing shared services.

  • Deploy blueprints to the cloud for a mobile-empowered workforce

  • Securely expose back-office workflows over REST with granular permissions

  • Out-the-box support Auth0, MongoDB and Elasticsearch

  • Easily extend out-the-box functionality with plug-ins
